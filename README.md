# SpeakWell — AI English Speaking Coach

🌐 **Live:** https://speakwell-pi.vercel.app

Record yourself speaking English and get instant AI feedback on:
- Grammar corrections with explanations
- Filler word detection
- Fluency, clarity, and confidence scores
- Corrected and natural audio versions

## Tech Stack
Frontend: Next.js, TypeScript, Tailwind CSS (Vercel)
Backend: FastAPI, Celery, Redis (Railway)
AI: Groq Whisper, Groq Llama 3.3 70B
Storage: Supabase Storage + PostgreSQL
Auth: Supabase Auth
Infrastructure: Docker, GitHub CI
